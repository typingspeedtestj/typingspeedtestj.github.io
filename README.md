# ⌨️ Typing Speed Test

A modern, multilingual typing speed test application with progress tracking and achievements.

![Typing Speed Test](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🌐 Demo

[https://typingspeedtestj.github.io](https://typingspeedtestj.github.io)

## ✨ Features

- **4 Languages**: English, Ukrainian, Russian, German
- **3 Time Modes**: 30s, 60s, 120s
- **Live Statistics**: CPM, WPM, Accuracy, Errors
- **Progress Tracking**: Calendar, history, streaks
- **12 Achievements**: Unlock badges as you improve
- **Keyboard Shortcuts**: Full keyboard navigation
- **Offline Support**: Data saved locally

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Enter` | Start test |
| `Esc` | Reset / Close |
| `Tab` | Try again |
| `F1` | Help |
| `1-4` | Switch tabs |
| `Alt+1-4` | Switch language |

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/typingspeedtestj/typingspeedtestj.github.io.git
```

2. Open `index.html` in your browser

No build tools or dependencies required.
